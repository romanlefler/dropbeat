
# v50.0.0

## Features

- GNOME 50 support
- Clicking the album art now opens a fullscreen pop-up window
- Setting to allow or disallow album art fetching over the Internet. Defaults to allow.
- Popup on first-run describing how to install ImageMagick if it is not found

## Improvements

- Missing album art icon fits the menu better now
- Album art with transparency is handled better on backgrounds
- Non-square album art is now cropped to a square
- Better responsiveness on track change

## Bug Fixes

- Closing the currently active player now causes the app to show another one if one is playing instead of disappearing

# v49.0.0

Initial release with GNOME 48 and 49 support.
